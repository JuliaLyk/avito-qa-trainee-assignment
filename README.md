# Тестирование API сервиса объявлений (Avito QA )

Данный репозиторий содержит проект по автоматизации тестирования REST API для сервиса размещений объявлений. Проект выполнен в рамках тестового задания на позицию **QA Automation Intern**.

**Отчет о багах**: [BUGS.md](./BUGS.md) — документированные критические дефекты.
**Тест-кейсы**: [TESTCASES.md](./TESTCASES.md) — подробная логика проверок (позитивные и негативные сценарии).

## Технологический стек
**Python 3.13**
* **Pytest** — основной фреймворк для тестирования.
* **Requests** — библиотека для взаимодействия с HTTP API.
* 🛠 **UUID / Re / Random** — генерация тестовых данных и парсинг ответов.

## Развертывание и подготовка

### 1. Подготовка окружения
Рекомендуется использовать виртуальное окружение для изоляции зависимостей.

```bash
# Создание виртуального окружения
python -m venv .venv

# Активация (Windows PowerShell)
.venv\Scripts\activate

# Активация (Linux/macOS)
source .venv/bin/activate
```

### 2. Установка зависимостей
Все необходимые библиотеки зафиксированы в файле `requirements.txt`. Для установки выполните:
```bash
pip install -r requirements.txt
```
*(Если файла нет, установите вручную: `pip install requests pytest`)*

### 3. Запуск автотестов
Для запуска всех тестов с подробным отчетом в консоли:
```bash
pytest test_api.py -v
```

Для дебага и просмотра логов (включая ответы сервера через `print`):
```bash
pytest test_api.py -v -s
```
